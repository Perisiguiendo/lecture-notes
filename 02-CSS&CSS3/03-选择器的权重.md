##  选择器的权重

- CSS选择器的进制是256进制

|                       | 权重     |
| --------------------- | -------- |
| ！important           | infinity |
| 行间样式              | 1000     |
| id                    | 0100     |
| class \| 属性 \| 伪类 | 0010     |
| 标签 \| 伪元素        | 0001     |
| 通配符                | 0000     |
| 继承的样式            | 无权重   |
